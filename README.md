# Movies APP
1. Install dependencies

  ```
  $ npm install
  $ bower install
  ```
2. Run server

  ```
  $ grunt
  ```
3. Open a browser and go to [http://localhost:8080](http://localhost:8080)

## Requirements

 Please, go to [requirements](https://github.com/sjaurena/movies/blob/master/REQUIREMENTS.md) section.
